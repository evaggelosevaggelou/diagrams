🖧 Τοπικό Δίκτυο σε Windows 10/11

Οδηγός βήμα-βήμα για κοινή χρήση αρχείων και εκτυπωτή — κάντε κλικ σε κάθε βήμα για λεπτομέρειες.

Πρόοδος 0 / 6
1
Συνδέστε τους Υπολογιστές
Ethernet ή Wi-Fi — επιλέξτε τη μέθοδό σας
🔌 ▼
  • Συνδέστε κάθε υπολογιστή στον δρομολογητή με καλώδιο Ethernet (πιο γρήγορο και σταθερό).
  • Αν χρησιμοποιείτε Wi-Fi, βεβαιωθείτε ότι όλοι οι υπολογιστές είναι στο ίδιο δίκτυο.
  • Αποφύγετε «δίκτυα επισκεπτών» — δεν επιτρέπουν επικοινωνία μεταξύ συσκευών.
  • Αν έχετε πολλές συσκευές και λίγες θύρες, αγοράστε ένα μεταγωγέα δικτύου (switch) ~15–20€.
💡 Ethernet = μέχρι 1 Gbps. Wi-Fi 5 = ~300–600 Mbps. Για μεταφορά μεγάλων αρχείων, προτιμήστε καλώδιο.
2
Ρυθμίσεις Δικτύου Windows
Ενεργοποίηση Ανακάλυψης Δικτύου
⚙️ ▼
  • Πατήστε Windows + I → «Δίκτυο και Internet».
  • Κάντε κλικ «Κέντρο Δικτύου» → «Αλλαγή ρυθμίσεων κοινής χρήσης».
  • Στο «Ιδιωτικό» προφίλ: ενεργοποιήστε «Ανακάλυψη δικτύου».
  • Ενεργοποιήστε επίσης «Κοινή χρήση αρχείων και εκτυπωτών».
  • Πατήστε «Αποθήκευση αλλαγών». Επαναλάβετε σε κάθε υπολογιστή.
💡 Εναλλακτικά: Windows + R → πληκτρολογήστε control → «Κέντρο Δικτύου και Κοινής Χρήσης».
3
Δημιουργία Κοινόχρηστων Φακέλων
Επιλέξτε τι θα μοιραστείτε και με ποια δικαιώματα
📁 ▼
  • Δεξί κλικ στον φάκελο → «Ιδιότητες» → καρτέλα «Κοινή χρήση».
  • Κλικ «Κοινή χρήση» → προσθέστε «Όλοι» στη λίστα.
  • Ανάγνωση: βλέπουν και αντιγράφουν — δεν αλλάζουν τίποτα.
  • Ανάγνωση/Εγγραφή: πλήρης πρόσβαση — μπορούν και να διαγράφουν.
  • Κλικ «Κοινή χρήση» → «Τέλος».
⚠️ Ξεκινήστε πάντα με «Μόνο Ανάγνωση». Μην μοιράζεστε ολόκληρο δίσκο ή φακέλους συστήματος.
4
Πρόσβαση από Άλλον Υπολογιστή
Δύο τρόποι για να βρείτε τους κοινόχρηστους φακέλους
🔍 ▼
  • Μέθοδος Α: Windows + E → κλικ στο «Δίκτυο» → εμφανίζονται οι άλλοι υπολογιστές.
  • Μέθοδος Β: Windows + R → πληκτρολογήστε \\ΟΝΟΜΑ-PC ή \\192.168.1.10 → Enter.
  • Για να βρείτε το όνομα PC: δεξί κλικ «Ο Υπολογιστής μου» → «Ιδιότητες» → «Όνομα συσκευής».
  • Δεξί κλικ σε φάκελο δικτύου → «Αντιστοίχιση μονάδας δίσκου» για μόνιμη πρόσβαση ως Z: κ.λπ.
💡 Δεν εμφανίζονται άλλοι υπολογιστές; Ελέγξτε ότι όλοι έχουν «Ιδιωτικό» προφίλ δικτύου και όχι «Δημόσιο».
5
Κοινή Χρήση Εκτυπωτή
Εκτυπώστε από οποιονδήποτε υπολογιστή
🖨️ ▼
  • Στον υπολογιστή με εκτυπωτή: Ρυθμίσεις → «Bluetooth και συσκευές» → «Εκτυπωτές» → κλικ εκτυπωτή → «Ιδιότητες» → καρτέλα «Κοινή χρήση» → ✅.
  • Δώστε απλό όνομα χωρίς κενά (π.χ. EktypothesSaloni).
  • Στους άλλους υπολογιστές: Ρυθμίσεις → Εκτυπωτές → «Προσθήκη» → «Δεν βρίσκεται στη λίστα» → «Με βάση το όνομα».
  • Πληκτρολογήστε: \\ΟΝΟΜΑ-PC\ΟΝΟΜΑ-EKTYP → Enter → εγκατάσταση drivers αυτόματα.
💡 Ο υπολογιστής με τον εκτυπωτή πρέπει να είναι ανοιχτός για να εκτυπώνουν οι υπόλοιποι.
6
Αντιμετώπιση Προβλημάτων
Λύσεις για τα πιο συνηθισμένα ζητήματα
🛠️ ▼
  • ❌ Δεν βλέπω άλλους PC: Ελέγξτε ότι η Ανακάλυψη Δικτύου είναι ενεργή και το προφίλ είναι «Ιδιωτικό» σε όλους.
  • 🔒 Ζητά κωδικό: Εισάγετε το όνομα χρήστη + κωδικό του απομακρυσμένου PC. Χωρίς κωδικό λογαριασμό δεν επιτρέπεται σύνδεση.
  • 🐌 Αργή μεταφορά: Χρησιμοποιήστε Ethernet, κλείστε streaming εφαρμογές.
  • 🚫 «Δεν έχετε δικαίωμα»: Ελέγξτε δικαιώματα φακέλου (βήμα 3) και ότι το antivirus δεν μπλοκάρει.
  • 🖨️ Δεν βρίσκει εκτυπωτή: Βεβαιωθείτε ότι ο κεντρικός PC είναι ανοιχτός και η κοινή χρήση εκτυπωτή ενεργή.
⚠️ Μην απενεργοποιείτε εντελώς το τείχος προστασίας — προσθέστε μόνο εξαιρέσεις για κοινή χρήση αρχείων.
