Virtual disk types — click for details
Σταθερού Μεγέθους Fixed Size Γρηγορότερος Fastest
Δεσμεύει αμέσως όλον τον χώρο.
Allocates all space immediately.

Δημιουργώντας έναν σταθερό δίσκο 50GB, αμέσως δεσμεύονται 50GB από τον πραγματικό σας δίσκο — ακόμα κι αν ο εικονικός είναι εντελώς άδειος.

Ιδανικός για: εικονικές μηχανές, συστήματα που χρειάζονται σταθερή απόδοση, παραγωγικά περιβάλλοντα.

Creating a 50GB fixed disk immediately occupies 50GB on your real disk — even if the virtual disk is completely empty.

Best for: virtual machines, systems needing consistent performance, production environments.

Δυναμικής Επέκτασης Dynamically Expanding Έξυπνος Smart
Μεγαλώνει μόνο όσο χρειάζεται.
Grows only as needed.

Ξεκινά σχεδόν κενός και επεκτείνεται καθώς προσθέτετε αρχεία. Ένας δίσκος ορισμένος στα 100GB με 10GB περιεχόμενο καταλαμβάνει μόνο ~10GB στον πραγματικό δίσκο.

Ιδανικός για: αρχάριους, δοκιμαστικά περιβάλλοντα, εξοικονόμηση χώρου.

Starts nearly empty and expands as you add files. A 100GB disk with 10GB content uses only ~10GB of real disk space.

Best for: beginners, test environments, saving disk space.

Δίσκος Διαφοράς Differencing Disk Για ειδικούς Advanced
Αποθηκεύει μόνο τις αλλαγές.
Stores only changes.

Βασίζεται σε έναν "γονικό" δίσκο και καταγράφει αποκλειστικά τις αλλαγές που κάνετε. Ο γονικός δίσκος παραμένει πάντα ανέγγιχτος.

Ιδανικός για: πειράματα, snapshot-based backup, εργαστήρια πληροφορικής.

Based on a "parent" disk, it records only the changes you make. The parent disk always remains untouched.

Best for: experimentation, snapshot-based backup, IT labs.

💡 Συμβουλή: Αν ξεκινάτε τώρα, επιλέξτε VHDX Δυναμικής Επέκτασης. Εξοικονομεί χώρο, είναι πιο ασφαλής και συμβατός με όλες τις σύγχρονες εκδόσεις Windows. 💡 Tip: If you're just starting out, choose Dynamically Expanding VHDX. It saves space, is more resilient, and compatible with all modern versions of Windows.
Πώς να δημιουργήσετε τον πρώτο σας δίσκο — κάντε κλικ σε κάθε βήμα
How to create your first virtual disk — click each step
1
Ανοίξτε τη Διαχείριση Δίσκων Open Disk Management
Πατήστε Windows + X και επιλέξτε Διαχείριση Δίσκων. Εναλλακτικά, πατήστε Windows + R, πληκτρολογήστε diskmgmt.msc και Enter. Press Windows + X and select Disk Management. Alternatively, press Windows + R, type diskmgmt.msc and hit Enter.
2
Δημιουργία νέου εικονικού δίσκου Create a new virtual disk
Στο μενού Δράση (πάνω αριστερά) κάντε κλικ στο Δημιουργία και σύνδεση εικονικού σκληρού δίσκου. In the Action menu (top left), click Create and Attach VHD.
3
Επιλέξτε τοποθεσία και μέγεθος Choose location and size
Επιλέξτε πού θα αποθηκευτεί το αρχείο (π.χ. Επιφάνεια εργασίας) και ονομάστε το MyDisk.vhdx. Ορίστε μέγεθος 20–50 GB για αρχή. Choose where to save the file (e.g. Desktop) and name it MyDisk.vhdx. Set a size of 20–50 GB to start.
4
Επιλέξτε τύπο δίσκου Select disk type
Διαλέξτε VHDX ως μορφή και Δυναμική επέκταση για εξοικονόμηση χώρου. Πατήστε ΟΚ. Choose VHDX as the format and Dynamically expanding to save space. Click OK.
5
Εκκίνηση και αρχικοποίηση Initialize the disk
Ο νέος δίσκος εμφανίζεται ως "Άγνωστος". Κάντε δεξί κλικ → Εκκίνηση δίσκου → επιλέξτε GPT (προτεινόμενο για σύγχρονα συστήματα). The new disk appears as "Unknown". Right-click → Initialize Disk → choose GPT (recommended for modern systems).
6
Δημιουργία κατατμήσεων και χρήση Create partition and use it
Κάντε δεξί κλικ στον αδέσμευτο χώρο → Νέος απλός τόμος → ακολουθήστε τον οδηγό. Σε λίγα δευτερόλεπτα ο δίσκος εμφανίζεται στην Εξερεύνηση αρχείων! Right-click the unallocated space → New Simple Volume → follow the wizard. In seconds, the disk appears in File Explorer!
Πλεονεκτήματα & μειονεκτήματα
Advantages & disadvantages
ΠλεονεκτήματαAdvantages
  • Δοκιμές χωρίς κίνδυνο για τον κύριο δίσκο
  • Test without risk to your main disk
  • Εύκολη μεταφορά σε άλλον υπολογιστή
  • Easy transfer to another computer
  • Αντίγραφα ασφαλείας ολόκληρου συστήματος
  • Backup entire system environments
  • Χωρίς αγορά hardware
  • No hardware purchase needed
  • Υποστήριξη από τα Windows χωρίς εγκατάσταση
  • Built-in Windows support, no install needed
ΜειονεκτήματαDisadvantages
  • Δυναμικοί δίσκοι μπορεί να είναι ελαφρώς πιο αργοί
  • Dynamic disks may be slightly slower
  • Αποσύνδεση μετά από κάθε επανεκκίνηση
  • Disconnects after every reboot
  • Χαλασμένο αρχείο = χαμένα δεδομένα
  • Corrupted file = lost data
  • Ο σταθερός τύπος δεσμεύει χώρο αμέσως
  • Fixed type reserves space immediately
