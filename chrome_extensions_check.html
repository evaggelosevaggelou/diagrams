🔍 Έλεγχος Επεκτάσεων στο Google Chrome

Κάνε κλικ σε κάθε ενότητα για λεπτομέρειες

1️⃣ Πώς να ανοίξεις τις Επεκτάσεις
Τρόπος Α
Μενού Chrome
Μέσω του κεντρικού μενού ρυθμίσεων
1
Πάτησε ⋮ επάνω δεξιά στον Chrome
2
Επίλεξε «Επεκτάσεις»
3
Πάτησε «Διαχείριση επεκτάσεων»
Τρόπος Β
Γραμμή Διευθύνσεων
Ο πιο γρήγορος τρόπος
1
Πληκτρολόγησε στη γραμμή διευθύνσεων:
2
chrome://extensions
3
Πάτησε Enter ✓
2️⃣ Τι να ελέγξεις σε κάθε Επέκταση
🏷️
Όνομα
Το αναγνωρίζεις;
Αν δεν θυμάσαι να την εγκατέστησες, υπάρχει λόγος ανησυχίας. Αναζήτησέ το στο Google.
📋
Λειτουργία
Τη χρειάζεσαι πραγματικά;
Διάβασε την περιγραφή. Αν δεν εξυπηρετεί κάποιο συγκεκριμένο σκοπό για σένα, καλύτερα να τη διαγράψεις.
🔑
Δικαιώματα
Τι πρόσβαση ζητά;
Από τις «Λεπτομέρειες» δες σε τι έχει πρόσβαση. Μια απλή επέκταση δεν χρειάζεται πρόσβαση σε ΟΛΑ τα δεδομένα.
⭐
Αξιολογήσεις
Τι λένε οι άλλοι;
Πολλά αρνητικά σχόλια ή χαμηλή βαθμολογία στο Chrome Web Store είναι σαφές κακό σημάδι.
🚩 Προσοχή — Σημεία Κινδύνου
Άγνωστο ή περίεργο όνομα επέκτασης
Ζητά πρόσβαση σε ΟΛΑ τα δεδομένα σου σε κάθε ιστοσελίδα
Δεν έχει αξιολογήσεις ή έχει πολλές αρνητικές κριτικές
Μοιάζει με γνωστή επέκταση αλλά με παραλλαγή στο όνομα
Υπόσχεται «δωρεάν προστασία» ή «επιτάχυνση browser»
