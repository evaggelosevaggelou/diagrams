⚙️ Ενεργειακή διαχείριση Windows

  • Τα Windows "κοιμίζουν" την κάρτα Wi-Fi για εξοικονόμηση ρεύματος
  • Λύση: Device Manager → Wi-Fi card → Power Management → απενεργοποίησε το "Allow computer to turn off this device"
  • Είναι η πιο συχνή αιτία (70–80% των περιπτώσεων)

🔧 Παλιός / ασύμβατος οδηγός

  • Μετά από αναβάθμιση Windows ο driver μπορεί να μην είναι συμβατός
  • Λύση: κατέβασε τον τελευταίο driver από τη σελίδα του κατασκευαστή
  • Ή χρησιμοποίησε Device Manager → Update driver

📡 Παρεμβολές / κανάλι router

  • Πολλά δίκτυα στο ίδιο κανάλι δημιουργούν παρεμβολές
  • Λύση: άλλαξε κανάλι στις ρυθμίσεις του router (π.χ. 1, 6 ή 11)
  • Δοκίμασε 5GHz αν το laptop το υποστηρίζει

♨️ Φθαρμένη κάρτα Wi-Fi

  • Σε παλιά laptops η κάρτα μπορεί να έχει φθαρεί από θερμότητα
  • Σύμπτωμα: αποσυνδέσεις που χειροτερεύουν με τον καιρό
  • Λύση: αντικατάσταση κάρτας ή USB Wi-Fi adapter
