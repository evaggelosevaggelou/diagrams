🔌 Λύση 7: USB Wi-Fi Adapter
Παρακάμπτει πλήρως την παλιά κάρτα — κλίκ για λεπτομέρειες
Αγόρασε έναν USB Wi-Fi adapter (π.χ. TP-Link TL-WN823N ή Archer T3U). Συνδέσου μέσω USB και εγκατέστησε τα drivers. Παρακάμπτει εντελώς την ενσωματωμένη κάρτα, λύνοντας τα περισσότερα hardware προβλήματα.

⚙️ Λύσεις 1–6: Κλίκ για λεπτομέρειες

1
Επανεκκίνηση Router & Υπολογιστή
▼
Απόσυνδεσε το ρεύμα του router για 30 δευτερόλεπτα, μετά κάνε επανεκκίνηση και στον υπολογιστή. Περίμενε 2 λεπτά πριν ελέγξεις τη σύνδεση.
2
Ανανέωση IP / Flush DNS
▼
Άνοιξε CMD ως Διαχειριστής και τρέξε: ipconfig /release
ipconfig /flushdns
ipconfig /renew Επιλύει IP conflicts και κατεστραμμένη DNS cache.
3
Ενημέρωση / Επαναφορά Drivers Wi-Fi
▼
Διαχείριση Συσκευών → Προσαρμογείς Δικτύου → Κάρτα Wi-Fi → δεξί κλικ → «Ενημέρωση». Αν δεν βοηθήσει, «Κατάργηση» και επανεκκίνηση για αυτόματη επαναφορά.
4
Έλεγχος Power Management
▼
Διαχείριση Συσκευών → κάρτα Wi-Fi → Ιδιότητες → Διαχείριση Ενέργειας → αποεπιλέγεις «Να επιτρέπεται απενεργοποίηση για εξοικονόμηση ενέργειας» → Εφαρμογή.
5
Αλλαγή DNS Server (8.8.8.8)
▼
Ρυθμίσεις → Δίκτυο → Wi-Fi → IPv4 → Αλλαγή DNS: Κύριο: 8.8.8.8   Δευτερεύον: 8.8.4.4 Επιλύει αργούς/αναξιόπιστους DNS servers του ISP.
6
Network Reset (Windows)
▼
Ρυθμίσεις → Δίκτυο & Internet → Σύνθετες ρυθμίσεις → Επαναφορά δικτύου. Επαναφέρει όλους τους προσαρμογείς στις εργοστασιακές ρυθμίσεις. Θα χρειαστεί επανεισαγωγή κωδικών Wi-Fi.
