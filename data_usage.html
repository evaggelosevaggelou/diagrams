📊 Κατανομή Κατανάλωσης Δεδομένων

Netflix / YouTube · 38%
Streaming βίντεο
▼
Το streaming βίντεο είναι η μεγαλύτερη κατηγορία κατανάλωσης. Το Netflix σε Full HD καταναλώνει ~3GB/ώρα, ενώ το YouTube σε 1080p περίπου 2.5GB/ώρα. Μείωσε την ποιότητα αναπαραγωγής για εξοικονόμηση δεδομένων.
Cloud Sync · 25%
OneDrive / Google Drive
▼
Η αυτόματη συγχρονισμός αρχείων τρέχει στο παρασκήνιο και μπορεί να καταναλώνει σημαντικό εύρος ζώνης. Μπορείς να ορίσεις ώρες συγχρονισμού ή να περιορίσεις το upload/download bandwidth από τις ρυθμίσεις.
Updates · 18%
Windows / App updates
▼
Οι ενημερώσεις των Windows και των εφαρμογών γίνονται αυτόματα. Μια μεγάλη ενημέρωση Windows μπορεί να φτάσει τα 3–5GB. Πήγαινε στις Ρυθμίσεις → Windows Update → Σύνθετες επιλογές για να ορίσεις ώρες ή όριο δεδομένων.
Λοιπά · 19%
Browser, gaming, κ.λπ.
▼
Περιλαμβάνει web browsing, online gaming, email και άλλες εφαρμογές. Τα online παιχνίδια καταναλώνουν σχετικά λίγα δεδομένα (~50–150MB/ώρα), αλλά τα patches μπορεί να είναι πολλά GB.
