Τελικό βήμα

🖱️ Βήμα 1 — Δεξί κλικ στο Start

  • Κάνε δεξί κλικ στο κουμπί Start (κάτω αριστερά)
  • Θα εμφανιστεί ένα μενού με επιλογές συστήματος
  • Επίλεξε "Διαχείριση Συσκευών" (Device Manager)

⚙️ Βήμα 2 — Διαχειριστής Συσκευών

  • Ανοίγει το παράθυρο "Device Manager"
  • Βρες την κατηγορία "Network Adapters"
  • Κλικ στο βέλος για να την αναπτύξεις

📶 Βήμα 3 — Κάρτα Wi-Fi

  • Βρες την κάρτα Wi-Fi στη λίστα (π.χ. Intel Wireless, Realtek)
  • Κάνε διπλό κλικ ή δεξί κλικ → Properties
  • Ανοίγει το παράθυρο ιδιοτήτων της κάρτας

🔋 Βήμα 4 — Διαχειριστής Ισχύος

  • Πάτα την καρτέλα "Power Management"
  • Βλέπεις την επιλογή "Allow the computer to turn off this device to save power"
  • Αυτή η επιλογή κοιμίζει την κάρτα Wi-Fi αυτόματα!

✅ Τελικό βήμα — Αφαίρεσε το τικ → OK

  • Αποεπίλεξε (κλικ για να αφαιρεθεί το ✓) την επιλογή Power Management
  • Πάτα OK για να αποθηκευτεί η αλλαγή
  • Τελείωσε! Τα Windows δεν θα κλείνουν πλέον την κάρτα Wi-Fi αυτόματα
