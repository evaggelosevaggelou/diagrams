🦊 Έλεγχος Επεκτάσεων στο Mozilla Firefox

Κάνε κλικ σε κάθε ενότητα για λεπτομέρειες

1️⃣ Πώς να ανοίξεις τις Επεκτάσεις
Τρόπος Α
Μενού Firefox
Μέσω του κεντρικού μενού
1
Πάτησε ☰ επάνω δεξιά στο Firefox
2
Επίλεξε «Πρόσθετα και θέματα»
3
Μετάβαση στην ενότητα «Επεκτάσεις»
Τρόπος Β
Συντόμευση / URL
Ο πιο γρήγορος τρόπος
1
Πάτησε Ctrl+Shift+A (Win) ή Cmd+Shift+A (Mac)
2
Ή γράψε στη γραμμή διευθύνσεων:
3
about:addons + Enter
2️⃣ Τι να ελέγξεις σε κάθε Επέκταση
🔵
Ενεργές Επεκτάσεις
Ποιες τρέχουν τώρα;
Οι ενεργές εμφανίζονται με μπλε διακόπτη. Απενεργοποίησε όσες δεν χρησιμοποιείς αντί να τις αφήνεις ενεργές.
🔑
Δικαιώματα
Τι πρόσβαση έχει;
Έλεγξε αν η πρόσβαση που ζητά είναι λογική για αυτό που υποτίθεται ότι κάνει η επέκταση.
📅
Ημερομηνία
Πότε εγκαταστάθηκε;
Οι πρόσφατες εγκαταστάσεις που δεν θυμάσαι είναι συχνά ύποπτες. Έλεγξε τη λίστα ταξινομημένη κατά ημερομηνία.
🔄
Ενημερώσεις
Ενημερώνεται τακτικά;
Εγκαταλελειμμένες επεκτάσεις (χωρίς ενημερώσεις χρόνια) είναι επικίνδυνες γιατί έχουν κενά ασφαλείας.
💡 Συμβουλή για το Firefox
Το Firefox διαθέτει «Προτεινόμενες Επεκτάσεις» που έχουν ελεγχθεί για βασική ασφάλεια. Παρόλα αυτά, ακόμα και αυτές καλό είναι να τις ελέγχεις περιοδικά και να κρατάς μόνο όσες πραγματικά χρειάζεσαι.
