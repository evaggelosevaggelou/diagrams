Πλήρης Καθαρισμός του Browser

Κάνε κλικ σε κάθε ενότητα για λεπτομέρειες

🔧 Βήματα Καθαρισμού
1
Cookies & Προσωρινά
Διαγραφή δεδομένων περιήγησης
🔵 Chrome
▸
⋮ → Ρυθμίσεις → Απόρρητο και ασφάλεια
▸
«Διαγραφή δεδομένων περιήγησης» → Προηγμένες
▸
Επίλεξε όλα + «Όλη η χρονική περίοδος»
🦊 Firefox
▸
☰ → Ιστορικό → Απαλοιφή πρόσφατου ιστορικού
▸
Εύρος χρόνου: «Όλα» + τσεκ όλα τα πεδία
2
Επαναφορά Ρυθμίσεων
Για επίμονα προβλήματα
🔵 Chrome
▸
Ρυθμίσεις → «Επαναφορά ρυθμίσεων»
▸
«Επαναφορά στις αρχικές προεπιλογές» → Επιβεβαίωση
🦊 Firefox
▸
Βοήθεια → Πληροφορίες αντιμετώπισης προβλημάτων
▸
«Ανανέωση του Firefox» → Επιβεβαίωση
⚠️ Τι χάνεται με την Επαναφορά;
▼
❌ Χάνεις: Όλες τις επεκτάσεις, προσωπικές ρυθμίσεις, αποθηκευμένες φόρμες

✅ Κρατάς: Σελιδοδείκτες, αποθηκευμένους κωδικούς πρόσβασης
⚡ Γρήγορες Ενέργειες μετά τον Καθαρισμό
🔄
Επανεκκίνηση Browser
Κλείσε και άνοιξε ξανά τον browser μετά κάθε αλλαγή για σωστή εφαρμογή ρυθμίσεων.
🔒
Άλλαξε Κωδικούς
Email, τράπεζα, κοινωνικά δίκτυα. Ενεργοποίησε 2-factor authentication παντού.
🛡️
Σάρωση Antivirus
Εκτέλεσε πλήρη σάρωση με Windows Defender, Malwarebytes ή Avast Free.
📦
Έλεγξε Προγράμματα
Πίνακας Ελέγχου → Προγράμματα. Ταξινόμησε κατά ημερομηνία και διέγραψε ύποπτα.
⬆️
Ενημέρωσε Browser
Βεβαιώσου ότι ο browser σου είναι στην τελευταία έκδοση για μέγιστη ασφάλεια.
📅
Τακτικός Έλεγχος
Ένας έλεγχος επεκτάσεων τον μήνα είναι αρκετός για να διατηρείς καθαρό browser.
